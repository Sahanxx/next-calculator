# Next Calculator (Next.js + Tailwind)

A clean, responsive calculator built with Next.js (App Router) and Tailwind CSS—complete with keyboard support, memory buttons (MC/MR/M+/M−), and a collapsible calculation history.

- Live demo: https://sahanxx.github.io/next-calculator/
- Tech: Next.js, React, Tailwind CSS, TypeScript

---

## Features

- Core operations: +, −, ×, ÷
- Extras: percent (%), toggle sign (+/−), delete (DEL), clear (C)
- Memory: MC, MR, M+, M−
- History: up to 25 items, clickable results, clear history
- Keyboard support: numbers, operators, Enter/Backspace/Escape
- Responsive UI with Tailwind, dark theme
- Static export (optimized for GitHub Pages)
